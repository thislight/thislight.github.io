Dart : 高性能、包管理集成、语法优良的编程语言

Dart 是 Google 前几年推出的编程语言，到现在 SDK 版本号才刚刚刷到 1.x 。

优点

推荐 Dart 有几个理由

动态静态类型随意使用

Dart在这方面真是非常令人舒服。你可以在程序中随意使用动态类型和静态类型，可以做到很多纯静态类型下做不到的事，但又保留静态类型的优势（Dart的动态类型也是一种静态类型，名为dynamic）。Dart将静态类型作为一种语法糖，只在检查模式下生效。

良好的语法

Dart 语法有些近似与 Java 和 C++，但是添加了很多语法糖，使写程序时不再需要包括很多多余的代码，并且可读性也很高。

高性能虚拟机

DartVM 是我见过的性能最高的虚拟机。不仅DartVM运行时性能高，而且 Dart 初始化和解析文件的速度也很快，体现在Dart CLI下极高的启动速度。（比某Node高到不知哪里去了）

原生异步支持

Dart中带有异步支持的dart:async库。可以有效提高后端并发能力。（并且由于是自带的库，所以一般第三方库都会有适配）

dart2js

Dart2js 是 Dart 官方的项目，正所谓前后端通吃…
并且还有某A可用。（可是我是Vue党…）

好用的 Pub

Dart 的依赖管理器是 Pub，Pub的功能异常丰富，甚至可以往里面丢资源文件。而且日常管理什么的也很方便。（虽然偶尔抽风…）不过由于Dart是个小众语言，所以国内没见到有 pub.dartlang.org 的镜像。（Pub 提供了很方便的方式切换源）

缺点

Dart有一个最明显的缺点，有这个就够了。

**的生态

pub.dartlang.org 上的第三方库就几百个这样…有些库还是用不了的（不要问我是怎么知道的）。根本没有 Python/Ruby 那种能够带来”只要搜一下就有合适的库“的感觉的生态。而且经常会让你生不如死（围绕一个库写了半天代码结果发现这个库是坏的🌚）。
